Digital Library of Common Stamping Images.  
Please feel free to use the images to help you design patterns.  
To add to our library please send us either clear graphics or good pictures of the actual stamps in leather.
If possible please identify the origin of the tool with item numbers and size of possible.
The idea is to have a library of accurately sized images for design purposes.

The way to read the filenames below is as follows:
tlf is Tandy Leather Factory indicating where to buy the tool
6511-00 is the item number
basketweave is the category
pt406xpt25 is the size as in .406" x .25"

To download the images in CMX format - which allows them to be used in vector graphics software like 
Corel Draw, Adobe Illustrator, and Xara, please click here to get the zip file Leather Tools Zip
